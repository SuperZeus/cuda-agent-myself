# Phase 0 Checklist

This repository phase is complete when all of the following are true:

- Repository skeleton matches the V3 plan
- Task schema and workspace metadata schema are defined
- Standard workdir template exists
- Sandbox runner v1 exists with timeout and logging
- Environment snapshotting exists
- Observation compression protocol is documented
- Read-only evaluation policy is documented and enforced
- At least 10 standardized tasks exist
- Any registered task can run `compile`, `verify`, and `profile`
